# 김키링이의서랍

키링 전문 쇼핑몰 프로젝트

## 프로젝트 정보

- **프로젝트명**: kimkeyring
- **개발 기간**: 2025.01 ~
- **개발자**: ㄹㅁ

## 기술 스택

**Backend**
- Spring Boot 3.x
- Java 17
- Gradle
- MyBatis
- JWT, OAuth2

**Frontend**
- React 18
- Vite

**Database**
- MariaDB 12.x

## 프로젝트 구조
```
kimkeyring/
├── kimkeyring-backend/
├── kimkeyring-frontend/
└── docs/
```

## 주요 기능

- 상품 조회 및 검색
- 회원 인증 (JWT + OAuth2)
- 장바구니 및 주문
- 결제 연동
- 관리자 페이지

## 개발 진행

- Phase 1: 프로젝트 초기 설정
- Phase 2: 상품 관리
- Phase 3: 회원 시스템
- Phase 4: 주문 시스템
- Phase 5: 결제 연동
- Phase 6: 관리자 기능
